Celestial of Space Destroyer

This project is based on the game Space Invaders. Created a fully working game, with animations and leaderboard. Including a range of interactive GUI options. With the feedback and reviews given, I made changes to the game according to their needs. For example, at the start the game didn't have a pause menu, and with a lot of feedback requesting it, I took it into consideration, and implemented it. This turned out to quite useful, as players can pause the game and continue where they left from.

TicTacToe Game

A replication of the classic yet famous game, TicTacToe. It includes: a sleek and intuitive 3x3 grid-based interface, two-player gameplay with alternating turns, automation win and draw detection systems, real-time game state updates, customisable player marks, and a responsive design. The game is simple and easy to learn, but challenging to master.
